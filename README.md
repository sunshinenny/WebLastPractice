# WebLastPractice
## 大二下册web前端课堂教学内容以及期末考试练习题
### 学了什么
- 基本的DOM操作

  - 通过节点修改属性，类等
  - 实现基本动画
  - 实现基本计算
- 基本的jQuery操作
    - 选择元素
    - 简单的动画效果
- 基本的 HTML5 新表单内容

        - chart
        - validfrom
    - 包括了 canvas 学习
- 基本的 BoosTrapt
    - 栅栏系统
    - 导航栏
